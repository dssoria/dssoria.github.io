# dssoria.github.io

Sitio personal con un módulo de registro financiero autenticado.

## Ejecutar localmente

1. Inicia el servidor:
   ```bash
   python3 server.py
   ```
2. Abre `http://localhost:3000`.

## Módulo de registros

- URL: `http://localhost:3000/registros.html`
- Usuario por defecto: `admin`
- Contraseña por defecto: `admin123`

Puedes cambiar credenciales con variables de entorno:

- `APP_USERNAME`
- `APP_PASSWORD`
- `PORT`
